# 示例：史官长期记忆

## 用户提示

```text
调用 $codex-skill-army：请记住我的科研写作偏好：中文报告要先给研究背景，再给方法分类，最后给研究空白；Word 按我固定格式。
```

## 预期调度

- 史官：判断是否有长期复用价值。
- 监军：确认不含隐私、凭据或未授权内容。
- 总帅：写入前给出脱敏摘要并请求确认。

## 标准确认

```text
这条信息适合长期复用。我建议史官保存为：
“用户偏好：中文科研报告结构为研究背景、方法分类、研究空白；Word 输出遵守用户固定排版规范。”
是否写入长期记忆？
```

## 验收重点

- 不自动保存原始文件或聊天隐私。
- 只保存用户确认的摘要。
- 读取记忆时说明来源是历史偏好，不替代当前要求。
