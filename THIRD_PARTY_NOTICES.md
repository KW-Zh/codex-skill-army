# 第三方说明

本项目是一个 Codex skill 调度协议，不重新分发以下外部项目源码。

| 项目 | 用途 | 许可证/说明 |
| --- | --- | --- |
| agentmemory | 可选史官长期记忆 | Apache-2.0，以其仓库和 npm 包声明为准 |
| CodeGraph | 可选代码图谱工部能力 | 以其项目许可证为准 |
| Codex skill installer / skill creator | 安装和校验 Codex skills | 以本地 Codex 系统技能声明为准 |
| 用户本地领域 skills | 主将和副将能力来源 | 各自许可证和安全边界独立生效 |

贡献者不得把第三方项目源码、私有论文全文、数据集、凭据或未授权材料直接提交到本仓库。
