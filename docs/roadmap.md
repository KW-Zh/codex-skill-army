# Roadmap

`codex-skill-army` 采用 0.x 节奏迭代。路线图描述优先方向，不承诺具体发布日期。

## 0.2.x

- 强化科研路由地图，覆盖更多论文、Word、科研代码和仿真场景。
- 增强压力测试，减少关键词型假阳性。
- 增加安装脚本 dry-run 和公开文案审计。
- 改进 README、示例、issue 模板和贡献体验。

## 0.3.x

- 增加更多真实科研工作流样例。
- 完善缺兵征召评分矩阵。
- 增加 Codex skill 兼容性检查建议。
- 探索更细的科研专项角色：引用官、数据官、统计官、图表官、伦理合规官。

## 长期方向

- 让 skill 军团在不硬编码技能表的前提下，更稳地理解用户当前和未来安装的 skills。
- 提供更多端到端验证模板，但不把第三方工具源码纳入本仓库。
- 保持隐私、安全和低 token 成本优先。
