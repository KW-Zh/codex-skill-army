# 军需维护协议

军需官负责维护技能库和能力地图，默认只读。

## 体检项目

- skills 数量和主要分类。
- 每个 skill 是否存在 `SKILL.md`。
- 是否存在触发描述重复或边界不清。
- references/scripts/assets 是否悬空。
- 是否含隐私外发、凭据、批量整理、删除等风险。

## 维护原则

- 优先修中枢地图，不批量改写领域 skill。
- 不删除、不移动、不覆盖用户已有 skill。
- 对高风险候选只出审计报告，等待用户确认。
- 新增 skill 后必须运行 `quick_validate.py`。

## 能力地图分类

- 科研论文与文献。
- 文档办公与 Word/PDF/PPT。
- 科研代码、数据和可视化。
- 工程仿真、CAD、Abaqus。
- Web/前端/部署。
- 安全和质量。
- 元技能、编排、记忆和 MCP。
