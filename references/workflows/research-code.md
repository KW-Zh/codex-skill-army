# 科研代码库工作流

## 触发

用户要求理解大型代码库、改科研软件、复现实验、检查数据处理 pipeline、定位模型训练或分析脚本。

## 点将

- 工部：CodeGraph 优先用于结构、符号、依赖、影响面。
- 主将：代码语言或框架相关 skill。
- 副将：测试、数据、可视化、安全、文档。
- 监军：运行最小测试、复现实验或静态检查。

## 流程

1. 先确认项目入口、运行方式、数据要求。
2. 大型代码库优先用 CodeGraph 或已有索引。
3. 不在不了解数据含义时修改科研结论。
4. 改动后保留复现命令和随机种子说明。
5. 未能运行测试时，说明阻塞条件。

## 风险

- 数据泄露。
- 随机性导致结果不可复现。
- 单位、坐标系、样本筛选条件被误改。
- 结果图看似正常但统计逻辑错误。
